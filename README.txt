data - папка со всеми файлами (прим. картинки)

Constants.py - там записаны все значения которые используются в game.py
К примеру ширина и высота окна, начальные координаты героя, прописаны пути к картинкам и т.д.

game.py - основной файл с классами и телом игры


#### Для героя и/или карты создадим отдельный файл hero.py и опишем его класс
#### Это нужно для удобности перемещения (меньше скроллинга) и читабельности кода!
